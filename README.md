**Humble Cats Safari — Tourism Website**

A single-page tourism website for Humble Cats Safari, a boutique safari operator based in Arusha, Tanzania. Built with vanilla HTML, CSS, and JavaScript — no frameworks or dependencies required. Features a full-screen hero with an SVG savanna landscape, safari package listings with pricing, destination highlights, experience showcase, guest testimonials, and a contact/booking section. The design uses a warm, earthy color palette inspired by the Tanzanian wilderness, with custom animations and a fully responsive layout. To use, simply open `humble-cats-safari.html` in any browser, or host it on any static web server (GitHub Pages, Netlify, Vercel, etc.).
